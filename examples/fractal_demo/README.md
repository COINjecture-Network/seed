## ğŸ“Š Technical Details

### Mandelbrot Set

The **Mandelbrot set** is generated by iterating the equation:

```
z_{n+1} = z_nÂ² + c
```

Where:
- `z` starts at 0
- `c` is a complex number representing each pixel's position
- The pixel is colored based on how many iterations it takes to diverge

**Implementation highlights:**
- Resolution: 1920x1080 (HD)
- Complex plane bounds: x âˆˆ [-2.5, 1.0], y âˆˆ [-1.0, 1.0]
- Maximum iterations: 256
- Vectorized computation using NumPy for performance

### Julia Set

The **Julia set** uses a similar iteration:

```
z_{n+1} = z_nÂ² + c
```

But here:
- `z` starts at the pixel's position
- `c` is a fixed complex constant (default: -0.4 + 0.6i)
- Creates intricate, often symmetric patterns

**Implementation highlights:**
- Resolution: 1920x1080 (HD)
- Complex plane bounds: x âˆˆ [-1.5, 1.5], y âˆˆ [-1.5, 1.5]
- Maximum iterations: 256
- Parameter c can be modified for different Julia sets

### Deterministic Color Palettes

Colors are generated using GoldenSeed's entropy stream:

1. Initialize generator with seed offset
2. Generate 256 RGB colors from consecutive entropy chunks
3. Map iteration counts to colors using modulo operation
4. Same seed always produces identical color scheme

```python

for _ in range(seed_offset):
    next(generator)
```
